# common

このリポジトリは、文化と不平等科研で採取されたデータ分析のためのスクリプトを保存している。

tree構造は以下のようになっている。

├── DataSetUp
│   └── data
│       ├── Name_and_Label2.xlsx  ★2
│       ├── インタビュー協力者run.xlsx ★３
│       └── 現代日本の文化と不平等に関する社会学的研究_ローデータ.sav   ★1
├── README.md
├── common.Rproj
├── tools_MISC
└── 読むこと空間
    ├── Step03_2読むこと空間の生成b_revised.qmd
    
    
cloneするとcommonというdirに今、ご覧になっているものが展開される。ただし、cloneされるのは、rmdやqmdといった処理スクリプト部分だけであり、データ、関連表などはdirを生成し、そこにcopyする必要がある。

DataSetUp/dirの

★1　現代日本の文化と不平等に関する社会学的研究_ローデータ.sa が、調査データの本体である。
★2  Name_and_Label2.xlsx は、★1のlabelledSPSSに保存されている眺めの変数名と処理過程での利便性を考えてつけた短縮変数名の対応表である。これは、Excelの表なので、これを編集すれば以下ようにも修正が可能である。

★3　インタビュー協力者run.xlsx は、回答者の中のインタビュー対象者を、runIDpと個体番号の対応表としたものである。
